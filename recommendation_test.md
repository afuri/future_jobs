# 🎯 Концепция теста

Каждый ответ добавляет баллы в:

* **кластеры**
* **типы деятельности**
* **навыки**

Финальный score профессии считается так:

```
score = 
  0.5 * совпадение по кластеру +
  0.3 * совпадение по типу деятельности +
  0.2 * совпадение по навыкам
```

Это даст стабильные, понятные рекомендации.

# 🧮 Как считается ТОП-10

Для каждой профессии:

```
cluster_match = 1 если совпал основной кластер
activity_match = количество совпавших видов деятельности
skill_match = количество совпавших навыков

score = cluster_match*50 + activity_match*20 + skill_match*10
```

Сортируем → берём 10 лучших.
